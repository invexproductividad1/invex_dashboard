🔗 Compartir Dashboard

Este enlace contiene todos los datos del período embebidos y comprimidos. Cualquiera que lo abra verá el dashboard listo, sin necesidad de cargar archivos.

💡 ¿Cómo publicar el dashboard permanentemente?
1. Guarda este archivo HTML en tu computadora (ya lo tienes)
2. Ve a github.com → Nuevo repositorio → Sube el HTML
3. En Settings → Pages → Branch: main → Guardar
4. Tu URL permanente será: tuusuario.github.io/nombre-repo
Desde ahí puedes compartir tanto la URL base (para cargar archivos) como los enlaces con datos.
⚠️ El enlace puede ser largo (depende del volumen de datos). Si tu mensajería lo trunca, usa WhatsApp Web, Telegram o correo. Los datos no se guardan en ningún servidor — todo está en el enlace.
